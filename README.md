# Supply Chain Blockchain DApp

A decentralized supply chain management application built on the Ethereum blockchain using Solidity smart contracts, Next.js, and Web3.js.

---

## Overview

The **Supply Chain Blockchain DApp** is designed to bring transparency, security, and traceability to supply chain processes.  
It records every stage of a productâ€™s journey on the blockchain, ensuring that data cannot be altered or manipulated.

This project demonstrates how blockchain technology can replace traditional paper-based and centralized supply chain systems.

---

## Key Benefits

- Transparency through blockchain-based records  
- Tamper-proof and immutable transaction history  
- End-to-end product traceability  
- Reduced fraud and manual verification  
- Decentralized and trustless system  

---

## Features

- Role-based access control (Owner, Supplier, Manufacturer, Distributor, Retailer)
- Product lifecycle management
- Supply chain stage tracking
- Real-time blockchain interaction using MetaMask
- Simple and responsive web interface
- QR code support for product verification
